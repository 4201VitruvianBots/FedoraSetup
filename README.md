# FedoraSetup

Files and instructions to set up a desktop with Fedora installed

- Dark Theme
- Colored prompt
- Enable RPM Fusion
- Add Flathub
- VSCode
- IntelliJ
- Slack
- GNOME Tweaks and GNOME Extensions (optional)
- Keybindings
    - Alt+Tab to Switch Window
    - Super+Tab to Switch Application
    - Ctrl+Alt+T to GNOME Terminal
- Set hostname
